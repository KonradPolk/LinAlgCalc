Hello Welcome to my Linear Algebra Calculator!

In this Linear Algebra Calculator you can do the operations of Matrix Addition and Matrix Multiplacition. You can add or remove rows/columns by clicking those respective buttons. The program will not allow you to create an invalid matrix combination and will always compensate by trying to make the other matrix scale with what you just changed. Leaving a input blank will result in the identity element for the corresponding calculation (0 for addition and 1 for multiplication). Anyways have fun and good luck studying Linear Algebra!
